# Day 1 Deliverables

[English](./day1-plan.md) | [中文](./day1-plan.zh-CN.md)

The Day 1 scaffold should provide:

- a shared analysis data contract
- mock analysis data
- a placeholder frontend entry page
- a Python analyzer entrypoint
- sample and output folders for local testing

End-of-day checklist:

- frontend files exist under `web/`
- analyzer files exist under `analyzer/`
- `samples/` contains a placeholder note for adding audio
- `output/` is ready for generated files
- the team can start Day 2 audio analysis work without restructuring
